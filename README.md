# Glammd Website

## Description
Glammd is a web application for booking beauty and wellness services. Clients can browse services, book appointments, and manage their reservations. Professionals can view their scheduled appointments and manage their availability.

## Features
- Client dashboard for current and past reservations.
- Marketplace to discover professionals and their services.
- Services page with detailed information for each professional.
- Booking page with customizable options (like makeup tone, color palette, skin type).
- Favorites page for clients to save preferred professionals.

## Technologies Used
- HTML, CSS, JavaScript
- Google Fonts: Tartuffo, Inter

## How to Use
1. Open `index.html` in a browser.
2. Sign up as a client or professional.
3. Navigate through dashboards, marketplace, services, and booking pages.
